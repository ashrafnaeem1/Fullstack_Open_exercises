# Exercise 0.5, Single page app diagram

```mermaid
sequenceDiagram
    participant browser
    participant server

    browser ->> server: GET https://studies.cs.helsinki.fi/exampleapp/spa
    activate server
    server ->> browser: HTML document, Status Code 200 (OK)
    deactivate server
    
    note right of browser: Browser then starts rendering the webpage. It reaches link to main.css. It then fetches main.css.

    browser ->> server: GET https://studies.cs.helsinki.fi/exampleapp/main.css
    activate server
    server ->> browser: CSS Stylesheet, Status Code 200 (OK)
    deactivate server

    note right of browser: Browser then fetches the linked javascript file

    browser ->> server: GET https://studies.cs.helsinki.fi/spa.js
    activate server
    server ->> browser: Javascript Script, Status Code 200 (OK)

    note right of browser: Browser executes the fetched javascript, which now requests for JSON data.

    browser ->> server: GET https://studies.vs.helsinki.fi/data.json
    activate server
    server ->> browser: JSON data file, Status Code 200 (OK)
    deactivate server

    note right of browser: Browser continues rendering the page.

```
